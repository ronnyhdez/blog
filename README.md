# Personal blog

# Blogging workflow:

 - Create an issue with the blog post idea
 - Create  branch
 - Use the `distill::create_post()` function
 - Include content
 - Save file
 - Use `build > Build Website` button
 - commit changes
 - push changes
 - create pull request
 - review pull request
 - accept pull request
 - check site
 
 Original Rmd files are in the `_posts` folder.
